# 🍬 Candy Hound 🐕  

## Logical Puzzle Game  
Stretch, shrink, and snack your way through tunnels as a cookie-hungry dachshund! 🐕🍪  

Collect keys, press buttons, and outsmart pesky moles in this quirky puzzle-arcade adventure where **your body is the tool**.
